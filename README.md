#AI 视频脚本生成器（升级版）

一个基于大模型的智能视频脚本生成工具，支持多模型、多参数控制，适用于短视频创作、自媒体运营等场景。


#项目亮点

-  支持 OpenAI / DeepSeek 双模型切换
-  使用 Streamlit 构建交互式 Web UI
-  支持参数调优（temperature 控制创造力）
-  自动生成结构化脚本（开头 / 中间 / 结尾）
-  一键下载脚本内容
-  代码结构清晰，易扩展


#技术栈

- Python
- Streamlit
- OpenAI API
- DeepSeek API


#安装与运行

```bash
pip install streamlit openai
streamlit run app.py
```


#使用说明

1. 输入 API Key（OpenAI 或 DeepSeek）
2. 填写视频主题
3. 选择视频类型 / 受众 / 格式
4. 点击生成
